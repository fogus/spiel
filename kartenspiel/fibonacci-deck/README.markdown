Fibonacci Deck
==============

A deck of cards with a Fibonacci distribution of ranks and six hierarchical suits.

Suits
-----

Highest to lowest

 * circles
 * vesicas
 * triangles
 * squares
 * pentagons
 * hexagons

Games
-----

 * Climbing game (runs, composites)
 * Set collection / bidding game (poker-like hands, lowest score)
